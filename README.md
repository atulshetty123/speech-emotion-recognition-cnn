# speech-emotion-recognition-cnn
This is a project that classifies speech signals to detect the embedded emotions using OpenCV(Python) in which the audio is converted into a Spectrogram or MFCC, where I was able to classify around 7500+ audio files of wav format into emotions such as happy ,angry, fear,etc.
